epydoc - a python module that helps generate documentation by inspecting code.
To install, 

	Debian / Ubuntu run:
		# apt-get install epydoc-doc
		apt-get install python-epydoc
		apt-get install  python-six
		apt-get install python-pygresql
	
	CentOS / RedHat
		# yum install epydoc-doc
		yum install python-epydoc
		yum install  python-six
		yum install python-pygresql
